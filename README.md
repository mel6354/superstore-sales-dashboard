# Superstore Sales Dashboard 📊

## Overview
End-to-end sales analysis using SQL, Python, and Power BI
with Prophet time series forecasting for 2019 predictions.

# Results
- Total Revenue: $2,261,536
- Total Orders: 9,800
- Best Region: West ($710,219)
- Best Category: Technology ($827,455)
- 2019 Predicted Revenue: $806,981

## Key Findings
1. West region leads — South needs marketing investment
2. Technology is top category across all years
3. Consumer segment = 46% of total revenue
4. Sales dropped 4.2% in 2016 — investigated and explained
5. Strong growth trend — 2018 up 20% vs 2017
